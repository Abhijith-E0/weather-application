# Weather App 🌤️

A responsive weather application built using **HTML, CSS, and JavaScript** that fetches real-time weather data using the **OpenWeather API**.

Users can search any city and get current weather details like temperature, weather condition, humidity, wind speed, and weather icons.

---

## Features

- Search weather by city name
- Real-time temperature data
- Dynamic weather conditions
- Humidity information
- Wind speed details
- Weather icons from API
- Responsive design
- Enter key search support
- Fast API-based data fetching

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeather API
